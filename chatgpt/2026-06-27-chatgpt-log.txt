A couple of recommendations based on your React experience:

* Wrap every page in a `.card` rather than placing content directly into `main`. It gives the app immediate visual structure.
* Add a `.container` utility if you ever need narrower pages (e.g. login/register):

```css
.container {
    width: min(100%, 1200px);
    margin: 0 auto;
}

.container-sm {
    width: min(100%, 480px);
    margin: 0 auto;
}
```

Then your login page becomes as simple as:

```html
<div class="container-sm">
    <div class="card">
        <h1>Login</h1>
        ...
    </div>
</div>
```

This is a very common layout pattern in Spring MVC/Thymeleaf applications and will give you a clean, modern look with minimal CSS.
